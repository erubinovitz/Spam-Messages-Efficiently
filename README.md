# Spam-Messages-Efficiently
Find the most efficient way to paste a message multiple times, aka the way to paste it with the fewest number of button presses

Input: a number of times the message must be pasted

Output: First, a string consisting of any number of three different characters: A, V, and C. A signifies ctrl A (select all), V signifies 
ctrl V (paste), and ctrl C (copy). Then, the number of characters in the string, aka the number of operations
